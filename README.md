# Covid_Projects
This is a collection of projects that retrieves Covid data, enters it into a database and displays the tables on a develpoment website.

## Project covid
### Project Overview
This is where all java code is developed. It contains code to update and correct errors in the database.

#### This project has dead code and working code.

## Project CovidPhpProject
### Project Overview
This is all the php files that extend the database and creates the json output for the front end.

#### This has all php files created not all are in use.

## Project CovedWebApplication
### Project Overview
This project has all the HTML and CSS files for the development website and bootstrap.

#### Not all files are in use.

## Project get_covid_data
### Project Overview
This project has the working java jar program that is running automatically on the application server.
